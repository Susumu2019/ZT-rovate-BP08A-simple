# ZT-rovate-BP08A-simple

## 概要
このプロジェクトは、M5CoreS3SE + PCA9685 + MPU6886 + サーボ + WS2812 16個を使ったロボット制御のサンプルです。
画面のボタン操作でサーボ角度を変更できる、シンプルな構成になっています。このロボットには傾きなどを検出するMPU6886が内蔵されています。慣れてきたらAIに聞いて使ってみてください。

## このプログラムの目的
このプログラムはシンプルに作られており、自分でマイコンソフトを作り上げていく事を目的としています。
必要な機能を少しずつ追加しながら、自分のロボット制御プログラムへ発展させてください。

## バイブコーディングの活用
VS CODEでバイブコーディングする環境を調べてセットアップする事を強くお勧めします。例えば「マイコンの画面によって動かすサーボはどこで指定していますか？」とチャットで質問するとどの行で指定してどの行でそれを実行しているか？なども説明してくれます。もちろんその修正もチャットでお願いする事ができます。

## 開発環境
- VS Code
- PlatformIO

## 書き込み時の重要な注意
ロボットへの書き込み時は、マイコン左側の USB 電源を抜く前に、必ずバッテリー電源を切ってください。
バッテリー電源が入ったままだと、ロボットが暴れる可能性があります。

## 書き込みとテスト手順
1. バッテリー電源を OFF にする
2. マイコン左側の USB 電源を抜く
3. USB 経由でプログラムを書き込む
4. 書き込み完了後、USB 電源を戻す
5. バッテリー電源を ON にしてテストする
